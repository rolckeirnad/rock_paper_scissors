# rock_paper_scissors
A 5 round Rock Paper Scissors game. 

The first task was to create a console game where user typed his choice and play against the computer. The result was shown in console.

The second task was adding a GUI for the game to be more user-friendly. This involved refactoring the actual code for make it work.

Check it out on live preview [here](https://rolckeirnad.github.io/rock_paper_scissors/)
